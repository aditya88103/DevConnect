# Assets Directory

This directory is for storing images, icons, and other static assets.

## Suggested Assets

- Logo files (SVG, PNG)
- Profile placeholder images
- Icon sets
- Background images
- Favicon

Currently, the application uses:
- SVG icons (inline in HTML)
- UI Avatars API for profile pictures (https://ui-avatars.com)
- CSS gradients for backgrounds

You can add custom assets here and reference them in your HTML/CSS files.
